# 운영체제 (Operating System)

## 책 정보
- **도서명**: 컴퓨터 밑바닥의 비밀
- **저자**: 크리스챤 알메이다, 제임스 쿠로스
- **출판사**: 한빛미디어

## 목차

### 1장: 컴퓨터 내부의 언어 체계
- 이진수, 십육진수
- 문자 인코딩
- 비트와 바이트

### 2장: 전자 회로의 조합 논리
- 불 대수
- 논리 게이트
- 가산기

### 3장: 메모리와 디스크의 핵심
- 순차 논리
- 메모리 조직
- 저장 장치

### 4장: 컴퓨터 아키텍처
- CPU 구조
- 명령어 집합
- 파이프라이닝

### 5장: 컴퓨터 아키텍처와 운영체제
- 시스템 콜
- 프로세스 관리
- 메모리 관리

### 6장: 입출력과 네트워킹
- I/O 장치
- 네트워크 프로토콜
- 통신 방식

### 7장: 데이터 구조와 처리
- 자료구조 기초
- 알고리즘
- 데이터 처리

### 8장: 프로그래밍 언어의 동작 원리
- 컴파일러
- 인터프리터
- 메모리 관리

2. 운영체제의 핵심 개념 파악하기
3. 하드웨어와 소프트웨어의 상호작용 이해하기
