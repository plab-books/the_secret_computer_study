## 📅 스터디 일정
- **기간**: 2026년 2월 ~ 2026년 4월 (예정)
- **시간**: 매주 일요일 오후 10시
- **장소**: 온라인 (Discord or 구글독스)
- **진행 방식**: 주 1회, 정해진 범위로 진행

## 📚 학습 도서
- **도서명**: [컴퓨터 밑바닥의 비밀](books/operating-system.md)
- **저자**: 크리스챤 알메이다, 제임스 쿠로스
- **출판사**: 한빛미디어

## 📋 스터디 규칙

### 1. 사전 학습 📖
- 매주 정해진 범위를 읽고 개인 정리 파일 작성
- `weeks/weekXX/본인이름.md` 형식으로 작성
- 최소 다음 내용 포함:
  - 주요 개념 정리
  - 토론 질문에 대한 답변
  - 추가 학습 내용 (선택)
  - 궁금한 점

### 2. 스터디 진행 🎯
- 각자 학습한 내용 발표 (10분)
- 토론 및 질의응답 (30분)
- 다음 주 범위 확인 및 질문 선정 (10분)

### 3. 출석 및 벌금 ⚠️
- **출석**: 주 1회 필수 참여
- **지각**: 10분 이상 늦을 시 지각 처리
- **결석**: 사전 공지 없이 불참 시 벌금 5,000원
- **과제 미제출**: 벌금 3,000원
- 벌금은 스터디 종료 후 회식비로 사용

### 4. 커밋 규칙 💻
```
주차 폴더 생성: mkdir weeks/weekXX
학습 내용 작성: touch weeks/weekXX/이름.md
커밋 메시지: "docs: Add weekXX 이름 study notes"
```
