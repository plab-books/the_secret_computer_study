# Week 02 - 기현

## 📚 학습 내용 정리

### 1. 불 대수 (Boolean Algebra)
- **AND**: 모든 입력이 참일 때만 참
- **OR**: 하나 이상의 입력이 참이면 참
- **NOT**: 입력을 반전
- **XOR**: 입력이 서로 다를 때 참

### 2. 논리 게이트
- 트랜지스터를 조합하여 논리 연산을 수행하는 기본 회로
- NAND, NOR 게이트로 모든 논리 연산 구현 가능
- 실제 CPU는 수십억 개의 트랜지스터로 구성

### 3. 가산기 (Adder)
- **반가산기 (Half Adder)**: 두 비트를 더함, 자리올림 출력
- **전가산기 (Full Adder)**: 세 비트를 더함 (이전 자리올림 포함)
- **리플 캐리 가산기**: 전가산기를 연결하여 다중 비트 덧셈

## 💭 토론 질문 답변

### 논리 게이트가 실제 CPU에서 어떻게 사용되나요?
논리 게이트를 조합하여 ALU(산술논리장치)를 만들고, 이를 통해 사칙연산, 비교 연산 등을 수행합니다. 또한 제어장치, 레지스터 등 CPU의 모든 구성요소가 논리 게이트로 만들어집니다.
